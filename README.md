# IRC Title 26 — Internal Revenue Code (USLM XML)

U.S. Code Title 26 (Internal Revenue Code) split into individual chapter files from the official USLM XML published by the [Office of the Law Revision Counsel](https://uscode.house.gov/).

**Source:** `xml_usc26@119-73not60` (current through Public Law 119-73, with some exclusions)
**Generated:** 2026-01-12 by USCConverter 1.7.2

## Structure

| Path | Description |
|------|-------------|
| `chapters/` | 72 XML files, one per IRC chapter |
| `meta.xml` | Document metadata |
| `preamble.xml` | Cross-reference tables (1939 Code vs 1986 Code) and title-level TOC |
| `subtitle-*.xml` | TOC and editorial notes for each subtitle |

### Chapter file naming

Files follow the pattern `st{subtitle}-ch{number}-{topic}.xml`, e.g.:

- `stA-ch001-normal-taxes-and-surtaxes.xml`
- `stD-ch042-private-foundations-and-certain-other-tax-exempt-organizatio.xml`
- `stF-ch068-additions-to-the-tax-additional-amounts-and-assessable-penal.xml`
